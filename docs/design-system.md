# 🎨 Pharmora Design System

## The future of pharmacy learning — open, collaborative, and accessible.

The Pharmora design system defines the visual language, components, and user experience principles of the platform.

---

# 🌱 Design Philosophy

Pharmora should feel:

- Modern
- Minimal
- Fast
- Trustworthy
- Educational
- Accessible

The interface should help users focus on learning.

---

# 🎨 Brand Identity


## Name

**Pharmora**

Pharmacy + Aurora

# 📱 Responsive Rules


Pharmora is mobile-first.

Must support:

- Phones
- Tablets
- Desktops


Rules:

- Avoid heavy animations
- Reduce loading size
- Keep touch targets large

---

# ⚡ Performance Principles

Avoid:

❌ Heavy libraries  
❌ Unnecessary animations  
❌ Large assets  


Prefer:

✔ Lightweight CSS  
✔ SVG icons  
✔ Optimized images  
✔ Browser-native features  

---

# ♿ Accessibility

Design should include:

- Clear contrast
- Keyboard support
- Readable text sizes
- Meaningful navigation

---

# 🧪 UI Personality

Pharmora should feel like:

A modern learning workspace.

Not:

- A file dumping website
- A complicated LMS
- A commercial dashboard
