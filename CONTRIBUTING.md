# 🤝 Contributing to Pharmora

Thank you for your interest in contributing to **Pharmora**.

Pharmora is an open-source pharmacy learning ecosystem built by students, educators, professionals, and developers.

Our mission:

> Make pharmacy education open, collaborative, and accessible.

Every contribution helps improve pharmacy learning for everyone.

---

# 🌱 Ways To Contribute


## 🎓 Educational Contributions

You can contribute:

- Study notes
- Topic explanations
- Diagrams
- Mnemonics
- MCQs
- Question banks
- Practical guidance
- Learning resources


Content areas:

- B.Pharm
- D.Pharm
- M.Pharm
- Pharm.D
- Pharmacy entrance exams


---

## 👨‍🏫 Teacher Contributions

Educators can share:

- Lecture notes
- Presentation ideas
- Teaching methods
- Course files
- Lesson plans
- Academic templates
- Classroom strategies


---

## 💻 Developer Contributions

Developers can help with:

- UI improvements
- Accessibility
- Performance optimization
- New features
- Bug fixes
- Documentation


---

# 🧪 Contribution Principles

Please ensure contributions are:

✔ Accurate  
✔ Educational  
✔ Clear  
✔ Respectful  
✔ Useful to the pharmacy community  


---

# 📚 Educational Content Guidelines


Good contributions:

- Explain concepts simply
- Mention references when needed
- Avoid misleading information
- Use original or permitted content
- Keep formatting clean


Avoid:

- Copyrighted material without permission
- Incorrect academic information
- Promotional/spam content

# 🎨 Design Philosophy

Pharmora UI should remain:

- Fast
- Clean
- Mobile friendly
- Accessible
- Minimal
- Modern


Avoid unnecessary complexity.

---

# 🔀 Pull Requests

Before submitting:

1. Keep changes focused
2. Test pages on mobile
3. Follow existing design style
4. Explain your changes clearly
