# 🔐 Pharmora Privacy Policy

## Privacy-first open pharmacy learning

Pharmora believes education should be open while respecting user privacy.

---

# Current Status

Pharmora is currently in:

🚧 Prototype Development

Current website:

- Static frontend
- No accounts
- No database
- No tracking system

At this stage Pharmora does not collect personal information.

---

# Privacy Principles

Pharmora development follows:

- Minimal data collection
- Transparency
- User control
- Open-source review
- Responsible data handling

---

# Information Collection

Future versions may require information for:

- User accounts
- Teacher profiles
- Contributions
- Community discussions

Only necessary information should be collected.

---

# Educational Contributions

Content submitted to Pharmora may include:

- Notes
- Discussions
- Teaching resources
- Corrections

Contributors should avoid sharing:

- Personal/private information
- Confidential documents
- Restricted materials

---

# Analytics

Pharmora aims to avoid invasive tracking.

Any future analytics should be:

- Privacy respecting
- Transparent
- Used only to improve learning experience

---

# Cookies

Current prototype:

No tracking cookies are used.

Future cookies/session storage may only support:

- Login sessions
- Preferences
- Platform functionality

---

# Data Ownership

Users should:

- Control their contributions
- Receive proper credit
- Understand how content is shared

---

# Open Source Transparency

Because Pharmora is open source:

- Code can be reviewed
- Improvements can be suggested
- Privacy issues can be reported

---

# Future Goals

Planned:

- Clear account controls
- Contribution management
- Data export options
- Secure authentication
